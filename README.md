## Предварительные требования

Перед запуском приложения необходимо подготовить окружение:

### 1. Установка и настройка LM Studio

1. Скачать LM Studio
2. Загрузить необходимые модели
3. Запустить локальный сервер и убедиться, что сервер запущен по адресу: `http://127.0.0.1:1234`

## Запуск приложения

### 1. Подготовка

Убедиться, что скрипт `start.sh` имеет права на выполнение:

```bash
chmod +x start.sh
```
Для запуска самого приложения нужно последовательно выполнить команды:

```bash
./start.sh
docker compose up --build -d
